# 01.conceitos linux

Destribuições mais famosas:

1. Ubuntu
A mais usada no mundo, fácil de instalar, grande comunidade. Ótima para desktop e servidores.

2. Debian
Muito estável, base do Ubuntu. Bastante usada em servidores e ambientes corporativos.

3. Fedora
Distribuição moderna com atualizações rápidas. Muito utilizada por desenvolvedores.

4. Red Hat / CentOS / Rocky / AlmaLinux
Família voltada para servidores e empresas. Red Hat é paga; Rocky e AlmaLinux são alternativas gratuitas.

# 02. Container Ubuntu  com Docker
Mão na massa: Vamos iniciar Ubunto em nossa maquina

https://hub.docker.com/

```bash
docker pull ubuntu # Baixa a imagem Ubuntu e salva de forma permanente na sua máquina (no cache do Docker), mesmo sem criar container.
docker pull ubuntu:25.10

docker run -it ubuntu # Se a imagem não existir localmente → o Docker baixa automaticamente.
docker run -it --name container-ubuntu-treino ubuntu


docker exec -it container-ubuntu-treino bash 
```

# 03. Atualizando Sistema e Instalando programas

```bash
# conhecendo as pastas linux
'''
Advanced Package Tool
apt = “loja de aplicativos” do Linux, só que via terminal.
Ele baixa, instala, atualiza e remove programas de forma automática.

'''


apt list                                 # Lista os pacotes disponíveis no repositório
apt update                               # Atualiza a lista de pacotes disponíveis
apt upgrade 
                                        
apt install python3 python3-pip         # Instala Python 3 e o pip
python3 --version                        # Mostra a versão do Python 3 instalada
pip3 --version                           # Mostra a versão do pip instalada
which python3                            # Mostra onde o executável python3 está localizado


apt install nano                      # Instala o editor de texto nano
nano teste.txt
cat tsts                                 # Lê e mostra o conteúdo do arquivo tsts
nano tsts                                # Abre o arquivo tsts para editar no nano
apt remove nano

whoami # usuario atual
echo $0 # pasta atual 
history
```

# 04.comandos de buscas 
```bash
ls
ls -1
cd
cd ..  # volta 1 diretório
cd /   # volta ao inicio 
pwd    # caminho atual 


find                                   # Busca tudo (arquivos e pastas) a partir da pasta atual
find -name 'Produto*'                  # Busca nomes que começam com "Produto"
find -name '*.csv'                     # Busca qualquer arquivo com extensão .csv
find -type f                           # Lista somente arquivos
find -type f -name 'Produto*'          # Lista arquivos que o nome começa com "Produto"
find -type d                           # Lista somente diretórios (pastas)
find -type d -name 'E-Commerce*'       # Lista diretórios que o  nome começa com "E-Commerce"

```
